## Why This Project Exists

Scientific data about wildlife should be accessible, engaging, and educational. The [BirdNET](https://birdnet.cornell.edu) bioacoustic monitoring network collects millions of bird observations globally, but this data often remains hidden in academic circles. These games bridge the gap between citizen science and public engagement, transforming real-time bird observations into interactive learning experiences.

By combining gameplay with authentic scientific data, we create opportunities for people to develop species recognition skills, understand biodiversity patterns, and connect with natureâ€”whether they're in a classroom, museum, or simply scrolling on their phone.

## Technical Philosophy

Built as single-file HTML applications optimized for accessibility and simplicity:

- **No installation barriers** - Works immediately in any modern browser
- **No build complexity** - Pure HTML/CSS/JavaScript for maximum transparency
- **Progressive Web App ready** - Install for offline play when desired
- **Responsive across all devices** - From smartphones to museum kiosks
- **Live data integration** - Real observations from the BirdNET API
